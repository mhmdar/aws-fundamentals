# Build on AWS: IAM, EC2, S3 & Beyond

A **4-hour hands-on workshop** for BZU IT students (Computer Engineering and Computer Science). The session blends short talks with guided labs so you build real AWS skills—not just watch slides.

**Build on AWS: IAM, EC2, S3 & Beyond** covers cloud fundamentals (IaaS, regions, shared responsibility), a Linux and Git ramp-up, and core services: **IAM**, **S3**, **EC2**, **ALB**, **RDS**, and **SQS**. Most of the time is spent in the console and CLI: verify your lab identity, host a static site on S3, run a web server on EC2, optionally add a load balancer, then tear everything down safely. Work in pairs—one partner on the AWS Console, one on the terminal.
